#Prerequisites Software

```
1.Vagrant
2.Virtualbox 
3.Ansible
```

#Deployment Steps
```
1. Download or clone the gitHub Repository..

$git clone https://github.com/mmjyothi/TestSains/

2. Change Directory 

$cd TestSains

$vagrant up

2. After the vagrant up is complete you can access to application at the below url

http://10.0.15.11/
```

#Redeployment Steps
```
$ vagrant destroy -f

```
